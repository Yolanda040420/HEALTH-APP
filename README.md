# HEALTH-APP
面向清华大学师生的综合性健康管理小程序
## 项目简介
健康助手是一款专为清华大学师生设计的健康管理应用，帮助用户科学管理饮食、记录锻炼，实现个人健康目标（减脂、维持、增肌）。结合校内食堂菜品数据，提供个性化的智能用餐推荐。
## 主要功能
###个人页面
个人健康档案管理（身高、体重、年龄、性别）
健康目标设置与追踪（减脂/维持/增肌）
当日卡路里概览显示
###用餐界面
基于校内食堂菜品的智能推荐
多维度筛选（口味、素食/清真、热量、蛋白质、价格、就餐时段）
菜品收藏、黑名单和评分功能
###饮食与锻炼记录
三种饮食记录方式：食堂菜品库、通用食物搜索、手动录入
锻炼记录与热量消耗计算
每日热量摄入与消耗统计
###论坛咨询
用户交流论坛
健康资讯与健身指南
换季提醒和过敏原提示
##使用说明
1、注册登录: 使用微信账号登录
2、完善资料: 在个人页面录入身高、体重等基本信息
3、设置目标: 选择减脂、维持或增肌目标
4、获取推荐: 使用"今日推荐"功能获取个性化用餐建议
5、记录数据: 每日记录饮食和运动情况
6、社区交流: 在论坛分享经验和获取健康资讯
##免责说明
本系统仅作为健康管理辅助工具使用，不提供医疗诊断功能。所有健康建议基于当前学界研究成果，用户应结合个人实际情况谨慎使用。
